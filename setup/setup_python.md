

# Environnement virtuel avec pyenv


## Création
Dans le dossier contenant le code du cours (par exemple le dossier de ce repository) : 
```bash
<votre-chemin>/.pyenv/versions/3.11.2/bin/python3 -m venv env
```

## Activation

```bash
source env/bin/activate
```
